# Reflexão sobre Trabalho em Equipa e Aprendizagem de MongoDB/NoSQL

## Trabalho em Equipa

Ao longo deste projeto tivemos a oportunidade de trabalhar em grupo e isso permitiu‑nos desenvolver competências essenciais para o nosso futuro profissional. A colaboração entre os dois foi fundamental para conseguirmos organizar tarefas, esclarecer dúvidas e resolver problemas de forma mais rápida e eficiente.

Aprendemos a:

- **Comunicar de forma clara** para evitar mal‑entendidos e garantir que todos estavam alinhados.
- **Distribuir tarefas de forma equilibrada**, aproveitando os pontos fortes de cada membro.
- **Apoiar‑nos mutuamente** quando surgiam dificuldades técnicas ou decisões importantes.
- **Gerir tempo e prioridades** para cumprir prazos e manter o projeto organizado.
- **Resolver problemas em conjunto**, discutindo soluções e tomando decisões informadas.

Este trabalho reforçou a importância da cooperação e mostrou‑nos que, quando cada pessoa contribui com o seu melhor, o resultado final é muito mais sólido.

---

## Importância de Aprender MongoDB e NoSQL

Aprender MongoDB e bases de dados NoSQL foi uma parte essencial deste projeto e trouxe‑nos várias vantagens para o futuro:

- **Compreendemos modelos de dados flexíveis**, que são muito usados em aplicações modernas.
- **Aprendemos a trabalhar com documentos**, uma abordagem diferente do modelo relacional tradicional.
- **Percebemos como otimizar consultas com índices**, algo fundamental para aplicações reais.
- **Ganhámos experiência prática com agregações**, pipelines e estruturas típicas de redes sociais.
- **Ficámos preparados para tecnologias usadas no mercado**, especialmente em aplicações web, mobile e sistemas distribuídos.

MongoDB é amplamente utilizado em empresas que lidam com grandes volumes de dados, microserviços e aplicações em tempo real. Esta aprendizagem dá‑nos uma base sólida para projetos futuros e aumenta a nossa preparação para o mercado de trabalho.

---

## Conclusão

Este projeto permitiu‑nos crescer tanto a nível técnico como a nível de trabalho em equipa. A experiência com MongoDB/NoSQL e a colaboração entre os dois foram fundamentais para alcançarmos um resultado final consistente, organizado e funcional.
